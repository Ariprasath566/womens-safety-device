# Women's Safety Device Prototype

## Overview
The Women's Safety Device Prototype is a personal safety project developed to provide an immediate self-protection and alert mechanism during emergency situations. The prototype focuses on electronic circuit design, hardware integration, and testing.

## Objectives
- Develop a compact personal safety device.
- Implement a trigger-based activation mechanism.
- Study electronic circuit integration and hardware prototyping.
- Improve practical skills in circuit debugging and testing.

## Components Used
- High-Voltage Electronic Circuit Module
- Switch/Trigger Mechanism
- Battery Supply
- Connecting Wires
- Prototype Enclosure

## Working Principle
1. The device remains in standby mode during normal operation.
2. When the trigger switch is activated, the electronic circuit is energized.
3. The system generates the intended safety response.
4. The device can be switched off after use.
